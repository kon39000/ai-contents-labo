No.05
— 隔週×6回で、ゼロイチまで一本道 —

「聞いて終わり」にしない、
各回、必ず1歩進むセミナー。

初回：2026年 4月27日(月) 12:00〜
そのあと、隔週で全6回／90日。アーカイブもあとから視聴できます。

  • 第1回 · 土台をつくる
    「自分には何もない」を終わらせる
    作ってこなかった時間も、作っても売れなかった時間も、ぜんぶ素材。
    自分の経験を、もう一度、自分で定義し直す時間です。
    • これまでの棚卸し
    • 感謝されたことを思い出す
    • 自分の経験の再定義
    WORK 感謝されたことを、5つ書き出す。
  • 第2回 · 価値の設計
    あなたの価値を「言葉」にする
    抽象的な「強み」を、「誰に、どんな変化を届けるか」まで一気に具体にします。
    少し前の自分を思い出すと、ちゃんと届けたい相手が見えてきます。
    • 誰に届けるか
    • どんな変化を届けるか
    • 少し前の自分を定義する
    WORK 「誰に、どんな変化を届けるか」を、1文で書く。
  • 第3回 · 発信を整える
    「発信しても反応がない」を抜け出す
    プロフィール、発信テーマ、刺さる言語化。
    「なんとなく書いていた」を、ここで卒業します。
    • プロフィール設計
    • 発信テーマの選び方
    • 刺さる言語化
    WORK Xプロフィールを書き換える／固定ポストを作る。
  • 第4回 · 商品にする
    価値を「商品」にする
    ここが、この90日でいちばん大事な回です。
    小さく作る・完璧は捨てる。その上で、あなた専用の商品案を決めます。
    • 商品の種類（note・Brain・テンプレ・相談 など）
    • 小さく作る考え方
    • 完璧を捨てる
    WORK 商品案を3つ出す → 1つに決める。
  • 第5回 · 売る怖さを解く
    「売るのが怖い」をなくす
    多くの人がここで止まります。
    だからこそ、「小さく売る設計」と「発信から販売までの導線」を、丁寧にほどいていきます。
    • なぜ売れないのか
    • 小さく売る設計
    • 発信 → 販売の導線
    WORK 100文字で、販売文を書く。
  • 第6回 · ゼロイチ実行
    その場で、実際に売る
    最終回は、聞く回ではなく、「動く回」です。
    販売リンクを出す。SNSで告知する。ゼロイチを、この回の中で迎えに行きます。
    • 販売リンクを公開する
    • SNSで告知を出す
    • 初収益を取りに行く
    WORK その場で、販売投稿を出す。
ONE LINE · 一本の線
自分の価値→誰に届けるか→発信→商品→販売
売れない人の多くは、ここがバラバラです。
この6回の価値は、全部をちゃんと一本につなぐこと。

各回で、必ずアウトプットします。
書く・出す・投稿する。聞くだけの人には、一切しません。
